How to Install Kodi BondNow

Build on FireStick / Androdid

Now that you know what Kodi’s BondNow build is, let’s start
with the step-by-step installation guide for Kodi 18. I have broken
down the process into three simple parts so that you will easily
understand why we are performing these steps

Part 1: Turn on Unknown Sources for Third-party Installation

1. If you haven’t turned on Unknown Sources already, go to Settings > System > Add-ons > Unknown Sources. Now turn it on.

2. Click Yes when prompted.

Part 2: Add Build Source and Install BondNow Wizard


1. Now go back to Kodi home screen, and click the Settings Gear icon.

2. Click File manager.

3. Go to last option in the list Add source.

4. Click <None>.

5. Enter the URL https://archive.org/download/bondnow/bondnow.zip
and then click OK

6. clickOK

7. Now press the back button once on your device and click Add-ons

8. Click Install from zip file.

9. Now select the repo name bondnow.zip

10. Click plugin.video.bondnow.zip

11. Wait for a few seconds and you will get an Add-on installed notification
in the right-top corner of your screen.

12. You’ll receive a prompt, click Continue.

13. Another prompt will appear, click Build Menu.

Part 3: Install BondNow build from Wizard

1. You will see several Bondnow build. You can select to suit your Device

2. Select FRESH INSTALL (Erase, then Install New Build).
You can also select STANDARD INSTALL but it’s better to start
fresh and remove all the previous files.

3. Next, if you have selected FRESH INSTALL, you’ll come across
the following warning message, click Continue to confirm.

4. Now wait for the download process to complete. It could take
some time even with high internet speed.

5. Once downloading and installing is done, a prompt will appear.
Click Force Close to shutdown Kodi and apply the build changes.

6. Now re-open Kodi on your device to use the BondNow build

Wait for the BondNow build to auto-complete remaining steps
like finishing skin settings and building menu. You should leave it
idle for around 15 minutes to avoid any errors. BondNow build >is now installed on Kodi.

Part 4: Install or Delete BondNow Libraries

1. Go to setting select File manager

2. select $ Home on the left and select $ Option on the right

3. In $ Option hold down on Kodi and seiect Seiect All

4.Hold down on Kodi again and select Move to install or Delete

5.Select "Yes" to Confirm
